## Overview

Multi-language sentiment analysis engine that processes customer reviews from **Google, Zomato, Swiggy, Amazon, and social media** to extract actionable business intelligence.

## Features

- **Multi-Platform Scraping** — Google Reviews, Zomato, Swiggy, Instagram, Facebook
- **Sentiment Classification** — Positive / Negative / Neutral with confidence scores
- **Aspect-Based Analysis** — Identify what customers love/hate (food, service, price, ambiance)
- **Emotion Detection** — Joy, anger, sadness, surprise, fear, trust
- **Trend Tracking** — Sentiment over time with anomaly alerts
- **Competitive Analysis** — Compare your sentiment vs competitors
- **Multi-Language** — English, Hindi, Malayalam, Tamil, Kannada
- **Automated Reports** — Weekly sentiment digest with action items

## Models Used

| Model | Purpose | Accuracy |
|:---|:---|:---|
| BERT (fine-tuned) | Sentiment classification | 91.5% |
| spaCy | Entity extraction | 89.2% |
| Custom CNN | Aspect detection | 87.8% |
| XLM-RoBERTa | Multi-language support | 88.1% |

## Use Cases

- **Restaurants** — Track Zomato/Swiggy review sentiment, identify menu issues
- **Hotels** — Monitor Google/TripAdvisor reviews, improve guest experience
- **E-commerce** — Analyze product reviews, detect quality issues early
- **Brands** — Social media sentiment monitoring, crisis detection
